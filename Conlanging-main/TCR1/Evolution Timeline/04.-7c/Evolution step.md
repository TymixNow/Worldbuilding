# Word formation

|      | *a_    | *o_    | *i_    |
| ---- | ------ | ------ | ------ |
| N, a | *a_a ¹ | *o_a   | *i_a   |
| V    | *a_o   | *o_o ¹ | *o_o   |
| D    | *a_i   | *o_i   | *i_i ¹ |
| N, e | *a_e ² | *o_e ² | *i_e ² |
¹ strengthening sense 
² personal sense
